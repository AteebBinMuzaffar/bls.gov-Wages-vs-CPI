The data in this analysis was extracted from the https://www.bls.gov/ database using their public api.

The datasets used were:
CPI for All Urban Consumers 
https://beta.bls.gov/dataViewer/view/timeseries/CUUR0000SA0 

Wages and salaries for Private industry workers in All industries and occupations, 12-month percent change
https://beta.bls.gov/dataViewer/view/timeseries/CIU2020000000000A

The aim of the analysis is to see how the wages of private employees in the US changed with the CPI for 
each quarter.
The analysis ranges from 2014 Q1 to 2022 Q4.
